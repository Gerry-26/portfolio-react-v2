# Portfólio React Pronto

Este projeto foi criado como modelo para uma atividade de migração de portfólio para React.

## Como rodar no computador

1. Instale o Node.js.
2. Abra a pasta do projeto no terminal.
3. Rode:

```bash
npm install
npm run dev
```

## Como publicar no GitHub

1. Crie um repositório novo no GitHub.
2. Entre nele.
3. Clique em **Add file** > **Upload files**.
4. Envie todos os arquivos desta pasta.
5. Clique em **Commit changes**.

## Como publicar na Vercel

1. Crie uma conta na Vercel.
2. Faça login com GitHub.
3. Clique em **Add New** > **Project**.
4. Selecione o repositório enviado.
5. Clique em **Deploy**.

Depois disso, copie os dois links:
- link do repositório do GitHub
- link do projeto publicado na Vercel

Esses serão os links que você colocará na atividade.
